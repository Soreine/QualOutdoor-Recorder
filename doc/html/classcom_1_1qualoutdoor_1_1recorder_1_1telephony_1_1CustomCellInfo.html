Detailed Description

Implementation of ICellInfo using a Bundle to store data.

The most convenient way to construct a CustomCellInfo is to use the static method buildFromCellInfo(CellInfo cell) which take as argument a CellInfo and try to grab as much information as possible from it, trying to cast it to the different subclasses (CellInfoLte etc.).
