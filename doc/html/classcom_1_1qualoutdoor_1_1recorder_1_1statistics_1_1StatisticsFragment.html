StatisticsPagerAdapter com.qualoutdoor.recorder.statistics.StatisticsFragment.statisticsPagerAdapter
private

The android.support.v4.view.PagerAdapter that will provide fragments representing each object in a collection. We use a android.support.v4.app.FragmentStatePagerAdapter derivative, which will destroy and re-create fragments as needed, saving and restoring their state in the process. This is important to conserve memory and is a best practice when allowing navigation between objects in a potentially large collection.
