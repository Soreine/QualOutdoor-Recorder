Detailed Description

Extending Application in order to hold some global constants that can't be stored in the XML files, and giving global access to application resources.

It is now possible to access the resources easily from a static context as follows :

static Resources res = QualOutdoorRecorderApp.getAppResources();

Gaborit Nicolas
