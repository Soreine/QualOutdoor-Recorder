Detailed Description

This Handler is used to manage AsyncTask related to the database by sending messages. The possible actions are starting or stopping a record and requesting an upload of the local database.

Gaborit Nicolas

static

Message code for initiating a conversion and upload of the database. This message should always be sent along with the chosen protocol code as arg1. Example :

Message msg = recordingHandler.obtainMessage(
RecordingHandler.MESSAGE_UPLOAD_DATABASE, chosenProtocol, 0);
