\subsection{Detailed Description}
A class that encapsulate the data required for inserting a sample in the database.

An instance of this class contains \-:


\begin{DoxyItemize}
\item A latitude and longitude
\item A measure context (Group, User, M\-C\-C, M\-N\-C, N\-T\-C)
\item A sparse array of measured data \-: (C\-I\-D, R\-S\-S\-I, Call, Upload, Download)
\end{DoxyItemize}
