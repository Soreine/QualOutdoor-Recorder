\subsection{Detailed Description}
Extending Application in order to hold some global constants that can't be stored in the X\-M\-L files, and giving global access to application resources.

It is now possible to access the resources easily from a static context as follows \-:


\begin{DoxyCode}
\textcolor{keyword}{static} Resources res = QualOutdoorRecorderApp.getAppResources();
\end{DoxyCode}
